# 🌟 Brightest Stars - Data Visualization Project

This project explores and visualizes the brightest stars in the night sky using Python and Jupyter Notebook.  
It covers key parameters like **magnitude**, **distance**, and **constellation**, offering a visual look into celestial objects that light up our sky.

---

## 📊 Technologies Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook
- CSV Data Processing

## 📊 What It Does
- Imports star data
- Plots star brightness vs distance
- Highlights key stars with labels
- Uses matplotlib for visualization

## 📁 Files
- `brightest_stars_pythonproject.ipynb`: Jupyter Notebook containing the code
